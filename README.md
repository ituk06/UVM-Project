# UVM Ассемблер и Интерпретатор

## Все этапы выполнены:

### Этап 1: Промежуточное представление ✓
- Парсер YAML в промежуточное представление

### Этап 2: Генерация машинного кода ✓
- Преобразование в бинарный формат (5 байт на команду)

### Этап 3: Интерпретатор и память ✓
- Модель памяти и регистров
- Базовые команды: load_const, read_mem, write_mem
- Дамп памяти в XML

### Этап 4: АЛУ (команда abs) ✓
- Реализация команды abs()
- Вычисление абсолютного значения

### Этап 5: Тестовая задача ✓
- Поэлементное применение abs() к вектору длины 9
- Три примера с различными данными

## Использование:
```bash
# Собрать программу
python assembler.py программа.yaml программа.bin --test

# Выполнить программу
python interpreter.py программа.bin дамп.xml 0 100

# Запустить тестовую задачу
python final_test.py
